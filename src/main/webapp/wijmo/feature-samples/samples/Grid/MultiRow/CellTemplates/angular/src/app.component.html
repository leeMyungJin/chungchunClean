Use an ng-template element with the wjMultiRowCellTemplate directive to define a cell template. The content of the ng-template element defines the cell content. The cellType property specifies the type of the cells represented by the template. The cell local template variable contains an object with cell specific data, including the data item (item), row (row) and column (col) that the cell represents.

Note that column-specific templates should be defined as children of the corresponding wj-multi-row-cell component, while the others are defined under the wj-multi-row element.
