Logarithmic Axis
=================

The [FlexChart](https://www.grapecity.com/wijmo/api/classes/wijmo_chart.flexchart.html) control allows you to display data using logarithmic axis.

Use the [axis](https://www.grapecity.com/wijmo/api/classes/wijmo_chart.axis.html) [logBase](https://www.grapecity.com/wijmo/api/classes/wijmo_chart.axis.html#logbase) property to spread clustered data and improve the clarity of your charts without sacrificing accuracy.

Use the [axis](https://www.grapecity.com/wijmo/api/classes/wijmo_chart.axis.html) [format](https://www.grapecity.com/wijmo/api/classes/wijmo_chart.axis.html#format) property to automatically scale the labels and show thousands or millions instead of very large raw values.
