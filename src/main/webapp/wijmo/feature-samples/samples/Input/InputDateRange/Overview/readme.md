InputDateRange Control
======================

The [InputDateRange](https://www.grapecity.com/wijmo/api/classes/wijmo_input.inputdaterange.html) control extends the [InputDate](https://www.grapecity.com/wijmo/api/classes/wijmo_input.inputdate.html) control and configures it to allow editing date ranges.

You may specify custom date ranges that the user can pick from, or they can use the multi-month calendar in the drop-down to select date ranges with the mouse or keyboard.
