ComboBox with HTML Content
==========================

The [ComboBox](https://www.grapecity.com/wijmo/api/classes/wijmo_input.combobox.html) allows you to customize the display of the items in the drop-down list. This sample applies custom HTML to each item in the dropdown, providing more information to the end user and the selection.
